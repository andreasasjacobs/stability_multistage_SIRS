This repository contains the MATLAB codes corresponding to the manuscript "Stability Analysis of a Multistage Compartmental Model of Infection" published in Journal of Biological Dynamics. 

There are three MATLAB codes here: 

1.General_Multistage_SIRS.mlx
The output of this file is the positive coefficients for the candiddate Lyapunov Functions

2.MULTISTAGE_SIRS_SIMPLIFY.nb
Mathematica script to simplify the results obtained in MATLAB especially for the case of n>=3

3.Simulation_Multistage_n_2.mlx
This code give the simulation results for two-stage SIRS model.

4.Simulation_Multistage_n_3.mlx
This code give the simulation results for three-stage SIRS model.
